📋 How to set up AppScript (one-time)

1
Open your Google Form → Responses tab → click the Google Sheets icon (View in Sheets) → your response spreadsheet opens
2
In Google Sheets → click Extensions → Apps Script
3
Delete all existing code → Paste the AppScript code from the AppScript_GCU.gs file provided
4
Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → click Deploy
5
Copy the Web app URL → paste it above → click Save → click Test Connection ✅
