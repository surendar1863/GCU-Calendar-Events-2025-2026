📋 Setup Guide

1
Create a Google Sheet from your Excel file → File → Import → upload .xlsx
2
In that Sheet → Extensions → Apps Script → paste AppScript_PhotoReport_v2.gs
3
Run requestPermissions → Allow → then Deploy → New deployment → Web App → Anyone
4
Copy the Web App URL → paste above → Save → Test ✅
